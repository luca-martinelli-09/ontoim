Demographic Observationc back to ToC or Class ToC

IRI: https://w3id.org/ontoim/DemographicObservation

A demographic observation is an observation made on the population of an entity, which can be the municipality, an organization, an event... The value defines the quantity of people or families observed, while the demographic reference represents the characteristics of the people observed. The demographic observation has also time and space delimitation. An example can be the number of female population of the municipality in 2021, or the number of italian citizens for a specific geographical area in 2020.
has super-classes
activity
has temporal entity max 1 temporal entity
has demographic referenceop max 1
has spatial delimitationop max 1 spatial object
observation valuedp exactly 1
has sub-classes
Bookingsc, Citizensc, Civil Statusc, Employeesc, Membersc, Subscribersc, Touristsc
is in range of
has demographic observationop

This class represents a public event (e.g., a football match, a conference, a book presentation, etc.). A public event is defined as an event that takes places in a given place and that occurs at a certain time, organised/run/promoted by one or more agents with a specific purpose and in the interest of a general public. There exist different types of public events depending on their size and content.
